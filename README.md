# iOS-Chat-App

A one-on-one iOS chat application built with Firebase Authentication and Firestore.

## Features

- User registration and login with Firebase Authentication
- One-on-one real-time messaging between authenticated users
- Chat history with timestamps and automatic scroll-to-latest
- WhatsApp-style chat UI with left/right-aligned message bubbles
- Basic input validation and user-friendly error messages

## Tech Stack

- iOS (UIKit)
- Firebase Authentication
- Firebase Firestore
- Swift
